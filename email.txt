Dear Team,

We would like to inform you that we have completed the required set of APIs to ensure that our mapping infrastructure is fully aligned with HUA’s semantic map specifications and needs.

Below you will find a summary of the current API capabilities, along with representative examples.

API Features

The response of the GET endpoint is a fully compliant RFC-7946 GeoJSON.

The produced GeoJSON output can be validated and tested directly via geojson.io.

Full support of the SimpleStyle Specification (Mapbox v1.1.0 more here: https://github.com/mapbox/simplestyle-spec/tree/master/1.1.0).

Supported HTTP methods:

GET
PUT
PATCH (for a specific element via POI ID)
DELETE (for a specific element via POI ID)

All coordinate values are returned in the format: [longitude, latitude].

You may test the API by performing a GET request: https://crispres.com/wp-json/map-manager/v1/features

Supported Geometry Types

The API currently supports all major GeoJSON geometry types, including:

Point, MultiPoint, LineString, MultiLineString, Polygon, MultiPolygon, GeometryCollection

Representative examples for each geometry type are provided below.

Point:

{
  "geometry": {
    "type": "Point",
    "coordinates": [23.7348, 37.9755]
  },
  "properties": {
    "name": "Syntagma Square",
    "category": "Landmark"
  }
}

Multipoint:

{
  "geometry": {
    "type": "MultiPoint",
    "coordinates": [
      [23.7348, 37.9755],
      [23.7275, 37.9715],
      [23.7295, 37.9680]
    ]
  },
  "properties": {
    "name": "Central Athens Points",
    "description": "Syntagma, Monastiraki, Acropolis area"
  }
}

LineString:

{
  "geometry": {
    "type": "LineString",
    "coordinates": [
      [23.7330, 37.9730],
      [23.7320, 37.9725],
      [23.7310, 37.9720],
      [23.7300, 37.9715]
    ]
  },
  "properties": {
    "name": "Plaka Walking Route"
  }
}

MultiLineString:

{
  "geometry": {
    "type": "MultiLineString",
    "coordinates": [
      [
        [23.7330, 37.9730],
        [23.7320, 37.9725],
        [23.7310, 37.9720]
      ],
      [
        [23.7280, 37.9705],
        [23.7270, 37.9695],
        [23.7260, 37.9685]
      ]
    ]
  },
  "properties": {
    "name": "Multiple Walking Paths"
  }
}

Polygon:
⚠ Important Note for Polygon geometries:
The first and last coordinate pair must be identical in order to define a closed linear ring, as required by the GeoJSON specification.

{
  "geometry": {
    "type": "Polygon",
    "coordinates": [
      [
        [23.7295, 37.9685],
        [23.7315, 37.9685],
        [23.7315, 37.9705],
        [23.7295, 37.9705],
        [23.7295, 37.9685]
      ]
    ]
  },
  "properties": {
    "name": "Acropolis Buffer Zone"
  }
}

MultiPolygon:

{
  "geometry": {
    "type": "MultiPolygon",
    "coordinates": [
      [
        [
          [23.7295, 37.9685],
          [23.7315, 37.9685],
          [23.7315, 37.9705],
          [23.7295, 37.9705],
          [23.7295, 37.9685]
        ]
      ],
      [
        [
          [23.7355, 37.9750],
          [23.7370, 37.9750],
          [23.7370, 37.9765],
          [23.7355, 37.9765],
          [23.7355, 37.9750]
        ]
      ]
    ]
  },
  "properties": {
    "name": "Two Protected Areas"
  }
}

GeometryCollection (mixed types):

  "geometry": {
    "type": "GeometryCollection",
    "geometries": [
      {
        "type": "Point",

      {

          [23.7310, 37.9720]
        ]
      }
    ]
  },
  "properties": {
    "name": "Mixed Geometry Example"
  }
}
